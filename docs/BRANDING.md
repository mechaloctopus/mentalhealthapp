# Branding — Mended Light

MoodSignal is a **Mended Light** app (Mended Light is the studio/parent brand).

## Source
Brand assets live in the Mended Light Google Drive brand folder. The folder contains:
`Mended_Light_Logo_DEC2020.ai`, `Mended_Light_LogoRGB_DEC2020.ai`, "Mended Light Logo Suite",
PNG/JPG logo exports, a **Style Guide**, a **Fonts** folder, `Cinzel.zip`, `AireBoldPro` order,
and `ML YT Banner.png`.

## Typography
- **Display / headings:** **Cinzel** (classical, luminous serif) — adopted in-app for `Display`
  headings and the wordmark via `@expo-google-fonts/cinzel` (`font.display`).
- **Wordmark script:** **Aire Bold Pro** (Sudtipos) — a commercial calligraphic script used in the
  logo. Not embeddable without the licensed font file; the logo art carries it.
- **Body:** Inter (UI), Newsreader (serif quotes/lessons) — retained for readability.

## Color
The exact palette wasn't readable from the locked Drive files. Current direction (refined toward an
elegant "Mended Light" feel, pending exact hex values):
- Warm luminous **gold**: `#e8c98a` / deep `#caa765` (brand/display accents)
- Brand gradient: gold → teal (`#f0d6a0` → `#66e0ca`)
- Existing premium dark canvas + teal/coral/amber/lavender accent system retained.
